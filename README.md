
# Create Styleguide

Create styleguides with no build configuration.


## Installation
Install it with npm.
```
$ npm install -g create-styleguide 
```


## Usage
Begin by typing:
```
$ create-styleguide <project-directory>
$ cd <project-directory>
$ npm start
```


## Options
Use the `--help` option.
```
$ create-styleguide --help
```
